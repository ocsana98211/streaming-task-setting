# Стриминговый сервис — постановка задач разработке

## Контекст
Проект стримингового сервиса.  
В рамках работы выполнялась аналитическая проработка функциональности и постановка задач разработке для реализации новых возможностей системы.

## Цель проекта
Подготовить требования и задачи для разработки функциональности стримингового сервиса с учётом пользовательских сценариев и архитектуры системы.

## Роль
Системный аналитик

## Задачи аналитика
- Анализ предметной области стриминговых сервисов
- Сбор и структурирование требований
- Формирование функциональных требований
- Постановка задач разработке
- Проработка пользовательских сценариев
- Участие в проектировании архитектуры системы
- Проектирование пользовательских интерфейсов (wireframes)

## Артефакты
- Функциональные требования
- Постановка задач разработке
- Пользовательские сценарии
- Диаграммы архитектуры системы
- Диаграммы взаимодействия компонентов
- Wireframes пользовательских интерфейсов

## Результат
Подготовлен комплект аналитических материалов и задач, необходимых для реализации функциональности стримингового сервиса.  
Материалы могут быть использованы командой разработки для начала реализации.

## Статус
✔️ Задание выполнено и принято по результатам ревью
